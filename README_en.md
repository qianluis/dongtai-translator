# Dongtai Dialect Real-time Translator 🌊

> A massive corpus-based Dongtai dialect translation system with real-time translation and voice support

## Features

### Real-time Translation
- Mandarin ↔ Dongtai Dialect bidirectional translation
- 120+ translation rules covering 12 categories
- Voice input and speech output support

### Massive Corpus
- 304,138 high-quality dialect-Mandarin parallel sentences
- 7 major categories, 50+ subcategories
- 5-level difficulty system

### Dialect Dictionary
- 500+ dialect vocabulary entries
- 12 category classification
- Search functionality

## Quick Start

Open `dongtai-translator-v6.html` directly in a browser. No server required.

## Translation Examples

| Mandarin | Dongtai Dialect |
|----------|-----------------|
| 我们今天去这里看看行不行？ | 我俫今朝去咯块望望好弗好？ |
| 这个东西多少钱了？ | 咯个杲昃好多钱呃？ |
| 小孩很聪明了 | 霞子蛮蛮灵光呃 |

## Key Translation Rules

- Pronouns: 我们→我俫 / 你们→你俫 / 这里→咯块
- Particles: 吗→呃 / 了→呃 / 的→个
- Verbs: 知道→晓得 / 看→望 / 找→寻 / 玩→白相
- Time: 今天→今朝 / 明天→明朝 / 昨天→昨日子

## Files

- `dongtai-translator-v6.html` — Translator (single file)
