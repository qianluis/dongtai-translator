# 东台方言翻译技能

## 功能
将普通话翻译为东台话，或将东台话翻译为普通话。支持语音和文字输入。

## 东台话核心知识

东台话属**江淮官话泰如片**，不是吴语！

## 翻译规则

1. **否定词**: 不→弗, 没有→呒得, 不要→弗要
2. **语气词**: 了→呃, 吗→呃, 的→呃(结构助词时)
3. **疑问句**: 了吗→呃没, 吗→呃
4. **程度副词**: 很/非常/太→蛮
5. **时间词**: 今天→今呃, 明天→明呃, 昨天→昨呃
6. **代词**: 我们→我俦, 你们→你俦, 他们→他俦
7. **动词**: 洗→汏, 穿→着, 看→望, 找→寻, 说→讲
