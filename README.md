## 🎯 方言准确性说明

东台话属于**江淮官话泰如片**，不是吴语。

核心特征词：
- 否定词：**弗**（不）、**呒得**（没有）、**覅**（不要）
- 疑问词：**哪块**（哪里）、**什的**（什么）、**怎呃**（怎么）、**几钿**（多少钱）
- 特色词：**杲昃**（东西）、**来斯**（厉害）、**推板**（差）、**扎实**（实在）、**刷刮**（利索）
- 称谓：**爹爹**（爷爷）、**伢儿**（孩子）、**女将**（妻子）、**邻舍**（邻居）
- 动词：**汏**（洗）、**寻**（找）、**着**（穿）、**望**（看）
