# 东台方言实时翻译 🌊

> 基于海量语料的东台方言翻译系统，支持实时翻译、语音输入输出

## 功能特性

### 🔤 实时翻译
- 普通话 ↔ 东台话双向翻译
- 120+ 翻译规则覆盖人称、动词、时间、称谓等12大类
- 支持语音输入和语音朗读

### 📊 海量语料
- 304,138 条高质量方言-普通话对照语料
- 7大类 50+子类场景覆盖
- 5级难度体系

### 📖 方言词典
- 500+ 方言词汇
- 12大类分类检索
- 搜索功能

## 快速使用

直接打开 `dongtai-translator-v6.html` 即可使用，无需服务器。

### 翻译示例

| 普通话 | 东台话 |
|--------|--------|
| 我们今天去这里看看行不行？ | 我俫今朝去咯块望望好弗好？ |
| 这个东西多少钱了？ | 咯个杲昃好多钱呃？ |
| 小孩很聪明了 | 霞子蛮蛮灵光呃 |
| 你找什么呢？ | 你寻什的呃？ |
| 医生，我肚子很疼了 | 先生，我肚皮蛮蛮痛呃 |

## 核心翻译规则

### 人称代词
我们→我俫 / 你们→你俫 / 他们→他俫 / 这里→咯块 / 那里→嗰块

### 语气词（灵魂！）
吗→呃 / 了→呃 / 呢→哩 / 吧→啘 / 啊→噶 / 的→个

### 核心动词
知道→晓得 / 看→望 / 找→寻 / 玩→白相 / 说→讲 / 回家→家去 / 聊天→搭寡

### 时间词
今天→今朝 / 明天→明朝 / 昨天→昨日子 / 晚上→夜头 / 早上→早起

### 称谓
爷爷→嗲嗲 / 小孩→霞子 / 丈夫→男将 / 邻居→邻舍

### 特殊句式
是不是→阿是 / 要不要→阿要 / 好不好→阿好 / 行不行→来事弗来事
