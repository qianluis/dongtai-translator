# 东台方言实时翻译器

东台话 ↔ 普通话 双向实时翻译，支持语音输入输出。

## ✨ 功能特性

- 🔄 **双向翻译**：普通话 ↔ 东台话实时翻译
- 🎙️ **语音输入**：Web Speech API 连续语音识别
- 🔊 **语音朗读**：双语朗读
- 📚 **方言词典**：500+核心词汇，12类分类检索
- 📖 **语料浏览**：400K语料库精选，分类筛选+搜索
- 📱 **移动端友好**：单HTML文件，直接打开即用
- 💬 **常用短语**：快捷翻译按钮

## 🎙️ 语音识别说明

### 权限问题已修复
- v8.1修复了语音权限反复弹出的bug
- 只在首次使用时请求权限，之后自动复用
- 如果用户拒绝，不再反复弹出，而是手动引导去设置

### 东台话语音识别
- **核心限制**：Web Speech API / 微信语音识别底层是普通话模型
- **解决方案**：说东台话时请**放慢语速**，系统会自动将识别出的普通话转为东台话
- **示例**：你对着手机说"我俫今朝"→ 系统识别为"我们今天"→ 自动翻译回"我俫今朝"
- **备选**：使用拼音输入法手动输入方言词

### 微信小程序语音
需配置[微信同声传译插件](https://mp.weixin.qq.com/wxopen/pluginbasicprofile?action=intro&appid=wx6e2a088ae8371c11)：
1. 在小程序后台 → 设置 → 第三方设置 → 插件管理 → 添加 `wx6e2a088ae8371c11`
2. 重新编译即可使用语音识别+TTS朗读
